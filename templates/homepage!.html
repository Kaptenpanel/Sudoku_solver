Sudokusolver

This website lets you solve a sudoku puzzle automatically.

To use the solver, go to this website: https://qqwing.com/generate.html,

and put the output format as "one line".
Then, copy the whole line and paste it into the box below.


using this web application, you can represent it visually, and solve it automatically if you wish to do so.

empty spaces in the board will be represented as dots.
